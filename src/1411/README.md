'숌스럽다'를 판별하는 함수만 구현하면 아주 쉽게 풀리는 문제입니다.   
두 문자열 a[]와 b[]에 대해, b[i]의 알파벳이 a에 이미 대응되어 쓰인 적이 있는지 그렇다면 a에 대응되어 쓰인 문자는 무엇인지 저장하는 배열 alpha[]를 만듭니다.   
만약 alpha[x] = a라면 (문법에 맞지 않는 예시입니다) b의 알파벳 x가 a의 a에 대응되었음을 의미합니다.   
이렇게 만약 alpha[i]가 0이라면 갱신하고, 0이 아닌데 현재 저장되어있는 값과 a[i]가 다르면 0을 리턴하는 과정을 반복합니다.   
끝까지 걸리지 않았다면 리턴값이 1이 됩니다.   
이렇게 '숍스럽다'를 판별하는 함수를 구현하고, 모든 가능한 순서쌍에 대해 '숌스러움'을 판별하면 됩니다.   
이 때, 위에서 구현한 함수는 일방향으로 '숌스럽다'를 판별했으므로, 함수를 2번 사용하여 양방향으로 검사해야 합니다.   
전형적인 실버 2 정도의 문제입니다.
