처음 보면 굉장히 어려운 정수론 문제처럼 보이지만, 규칙을 발견하면 엄청나게 쉽습니다.   
만약 K가 9의 배수이면   
```
9999^2 = 99980001
27999^2 = 783944001
36999^2 = 1368926001
```
이런 규칙으로 (우변과 좌변의 S(n)값은 같습니다.) 풀어주면 됩니다.   
만약 K가 9로 나눈 나머지가 1이면   
```
19999^2 = 399960001
55999^2 = 3135888001
73999^2 = 5475852001
```
이런 규칙으로 풀어주면 됩니다.    
그 외의 경우에는 -1을 출력합니다.   
플레티넘 3 치고 쉬운 문제였습니다.
